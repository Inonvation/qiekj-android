# LightLife v1.3.0

## ✨ 新增
- 快捷方式功能，内置三个预设（淘宝取件码、拼多多取件码、学校教务系统），支持自定义编辑，长按卡片可添加到手机桌面，拿快递再也不用手忙脚乱了
- 新增保险模式开关，开启后完全禁用积分脚本，适合担心账号风险的用户
- 支持启动 App 后自动检测积分任务，未完成则自动执行，无需手动开启（默认关闭，可在设置中开启）
- 现已支持后台刷积分，无需停留在前台，通知栏展示任务执行进度
- 支持一键清除所有记录和日志，无需担心过度占用存储空间

## 🐛 修复
- 修复支付宝视频任务无限执行且不加积分的 bug
- 修复标签记录不生效的 bug
- 修复预设卡片排序后重置错乱的 bug
- 其他已知问题修复

## ⚡ 优化
- UI 排版和动画交互全面优化
